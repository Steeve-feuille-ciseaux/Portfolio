Lovable et Flutter

Lovable ne code pas encore en Flutter ni pour les applications mobiles natives (Android/iOS). Il est conçu pour créer des applications web en utilisant principalement des technologies comme :

  • React (front-end)
  • Tailwind CSS (style)
  • Supabase (base de données et authentification)
  • Stripe (paiements)
  • Node.js (côté serveur, parfois)

❌ Limitations actuelles :

  • Pas de génération d'apps Flutter, Swift, Kotlin ou React Native.
  • Pas d'accès direct aux fichiers du smartphone (stockage local, caméra, etc.) comme on peut le faire avec Flutter ou d'autres frameworks mobiles.

✅ Alternatives si tu veux du Flutter avec IA :

Si ton objectif est de créer une application mobile Flutter avec des fonctionnalités comme l'accès aux fichiers du téléphone, tu peux essayer :

  • Codeium ou Cursor : assistants IA dans des IDE (VS Code, Android Studio).
  • GPT-4 (comme ici) : tu peux me décrire l’app, et je peux t’aider à générer du code Flutter.
  • Github Copilot : aide bien dans des projets Flutter si tu as déjà un environnement de développement installé.
