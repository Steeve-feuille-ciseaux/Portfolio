Résumé : Exécution d'un programme via clé USB sur Android et iPhone

1. Limitations générales

  • Android et iOS ne permettent pas l'exécution automatique de code externe.
  • Les systèmes mobiles bloquent tout autorun pour des raisons de sécurité.
  • Pas d'équivalent d'"autorun.inf" comme sous Windows.

2. Solution pour Android

  • Placer un fichier .apk sur la clé USB.
  • Ajouter un fichier LISEZMOI.txt pour guider l'utilisateur.
  • L'utilisateur doit autoriser l'installation d'apps depuis des sources inconnues.
  • Utiliser un gestionnaire de fichiers pour ouvrir et installer l'APK.

3. Solution pour iOS

  • Placer des fichiers médias ou une WebApp locale (HTML/CSS/JS).
  • Fichiers ouverts avec l'app Fichiers ou l'app du fabricant de la clé USB.
  • Pas de possibilité d'installation d'app, ni d'exécution de code natif.

4. Solution WebApp locale (Android & iOS)

  • Créer une page HTML interactive (WebApp) fonctionnant localement.
  • Inclure un fichier index.html dans la clé USB.
  • Permet lecture sans installation sur navigateur mobile.

5. Résumé comparatif

Plateforme Méthode Installation Exécution automatique
Android APK sur clé + guide Manuelle Non
iOS WebApp ou médias Lecture manuelle Non
Les deux WebApp locale Lecture immédiate Non

6. Structure recommandée à placer sur la clé USB

/ma-webapp/
│
├── index.html     ← Page d'accueil (déjà générée)
├── style.css      ← Feuille de style externe (optionnelle)
├── lisez-moi.txt  ← Instructions pour l’utilisateur
└── /media/        ← (facultatif) Vidéos, images ou autres fichiers liés

7. Exemple de fichier lisez-moi.txt

Ouvrez le fichier index.html dans votre navigateur.
Sur Android : utilisez un gestionnaire de fichiers pour y accéder.
Sur iPhone : utilisez l'app Fichiers ou celle de votre clé USB.
Aucune installation nécessaire.

8. Option complémentaire : QR Code de redirection

Pour simplifier l'accès ou proposer une version en ligne de la WebApp ou d’un guide :

  • Générez un QR code redirigeant vers un site web ou fichier hébergé (ex. GitHub, Google Drive, Dropbox).
  • Imprimez ou affichez le QR code dans un fichier image inclus sur la clé.
  • Exemple de lien : https://exemple.com/mon-webapp
