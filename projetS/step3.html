Comparatif en mettant l’accent sur les capacités d’accès au système du smartphone (comme accéder aux fichiers, faire du copier-coller, etc.)

Flutter (Dart)

  • Très fluide, moderne, UI personnalisable.
  • Accès au système via des plugins comme file_picker, clipboard, etc.
  • Idéal pour les apps performantes avec interaction native.

Exemple Flutter – Copier/Coller et Accès aux Fichiers

// pubspec.yaml dependencies :
clipboard: ^0.1.3
file_picker: ^6.0.0

import 'package:flutter/services.dart';
import 'package:file_picker/file_picker.dart';

void copyTextToClipboard(String text) {
  Clipboard.setData(ClipboardData(text: text));
}

Future pickFile() async {
  FilePickerResult? result = await FilePicker.platform.pickFiles();

  if (result != null) {
    String? filePath = result.files.single.path;
    print("Fichier sélectionné : \$filePath");
  }
}

React Native (JavaScript)

  • Basé sur React, très populaire, facile pour les développeurs web.
  • Utilise des modules comme react-native-fs et @react-native-clipboard/clipboard.

Exemple React Native – Copier/Coller et Accès aux Fichiers

// Installer les modules :
npm install @react-native-clipboard/clipboard react-native-fs

import Clipboard from '@react-native-clipboard/clipboard';
import RNFS from 'react-native-fs';

const copyToClipboard = (text) => {
  Clipboard.setString(text);
};

const listFiles = async () => {
  const files = await RNFS.readDir(RNFS.DocumentDirectoryPath);
  files.forEach(file => console.log("Fichier :", file.name));
};
